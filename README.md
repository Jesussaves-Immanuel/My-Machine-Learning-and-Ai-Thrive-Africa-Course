# 🤖 Thrive Africa: Machine Learning & AI Course

This repository houses my projects, assignments, and practical implementations for the **Machine Learning and AI Program** at [Thrive Africa](https://www.thriveafrica.co). The course focuses on building industry-ready skills in data science, predictive modeling, and AI ethics.

## 📌 Course Objectives
- Master **Data Wrangling** and Exploratory Data Analysis (EDA).
- Build and evaluate **Supervised** (Regression, Classification) and **Unsupervised** (Clustering) models.
- Explore **Deep Learning** and Neural Networks.
- Address **AI Ethics** and bias in algorithmic decision-making.
- Deploy ML models into real-world production environments.

---

## 📂 Repository Structure


| Folder | Description |
| :--- | :--- |
| `module-1-fundamentals/` | Basic concepts, Python for Data Science, and NumPy/Pandas basics. |
| `module-2-data-processing/` | Data cleaning, feature engineering, and visualization using Matplotlib/Seaborn. |
| `module-3-supervised/` | Linear Regression, Decision Trees, and Support Vector Machines. |
| `module-4-unsupervised/` | K-Means Clustering and Principal Component Analysis (PCA). |
| `capstone-project/` | Final project solving a real-world problem presented during Demo Day. |
